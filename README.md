**Run locally (not hosted yet)**

To run the project locally:
	•	Clone the repository
	•	Navigate to the cloned folder on your machine
	•	Ensure Python / Python3 is installed
	•	Start a local server:

python3 -m http.server 8000

	•	Open a browser and go to:

http://localhost:8000

**Commit conventions**
Use the following prefixes when committing to this project:
* feat – new feature
* fix – bug fix
* docs – documentation changes
* style – formatting changes (no logic changes)
* refactor – code changes without new features or bug fixes
* test – adding or updating tests
* chore – maintenance tasks (dependencies, config, etc.)
