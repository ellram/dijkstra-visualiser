# Graph Algorithm Visualizer

A basic visualizer for graph algorithms, with a focus on **Dijkstra's algorithm**.

## Features
- Visualizes Dijkstra's algorithm on custom graphs
- Step-by-step animation of shortest path calculation

## Technical Stack
- **Python** – core implementation
- **Matplotlib** – for visualization of graphs and algorithm steps

## Installation
1.  Clone the repository:
    git clone https://github.com/your-username/your-repo.git

2.	Navigate to the project folder:
    cd your-repo

4.	Install dependencies:
    pip install matplotlib

## Usage
Run the visualizer:
  python main.py

Modify the graph structure in the code to test different scenarios.
